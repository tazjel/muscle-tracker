# Open Hardware 3D Scanning Section

This directory contains research, hardware configurations, and shopping lists for building a professional-grade DIY human 3D scanning rig.

## Purpose
The goal is to provide a standardized hardware specification that works seamlessly with the `gtd3d` vision and skin verification engine.

- [SHOPPING_LIST.md](file:///C:/Users/MiEXCITE/Projects/gtd3d/openhardware/SHOPPING_LIST.md): Recommended parts for building a Raspberry Pi 5 / ESP32-S3 based rig.
- [RESEARCH_SUMMARY.md](file:///C:/Users/MiEXCITE/Projects/gtd3d/openhardware/RESEARCH_SUMMARY.md): Consolidated research on skin metrics, SSS, and specular highlights from Task 4.
