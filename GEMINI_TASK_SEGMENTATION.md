Our `bpy.ops.mpfb.add_rig()` call failed. But MPFB2 must have a way to add an armature with weight-painted bones.

**Find:**

- Check MPFB2 source code: https://github.com/makehumancommunity/mpfb2 — search for `rig`, `armature`, `weights`, `bone`

### 3. MakeHuman Proxy/Clothing Vertex Groups
MakeHuman uses "proxy" meshes for clothing that reference body vertex groups. These proxy definitions contain body part masks.

**Find:**
- Proxy vertex group files (usually in `data/proxymeshes/` or similar)
- The "body zones" or "body regions" used for proxy fitting — these define exactly which vertices are torso, arms, legs, etc.
- Face group definitions (which faces belong to which body part)
